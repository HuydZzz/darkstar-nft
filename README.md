## Overview

DarkStar is a premium NFT collection living on the [Stargaze](https://www.stargaze.zone) network. Each entity is a unique composition of hand-crafted layers — two distinct characters (Male & Female) set against six cosmic backgrounds. Holders gain governance weight across Cosmos-aligned protocols through IBC utility.

**Collection size:** 108 unique entities  
**Blockchain:** Stargaze (Cosmos Ecosystem)  
**Technology:** IBC · Inter-Blockchain Communication  
**Characters:** Male · Female  
**Backgrounds:** 6 variants

## NFT Generation

Images are generated by compositing 3 layers in order: **background → body → mask**

```bash
# Install dependencies
npm install

# Generate all 108 NFTs
node generate.js
```

Output:

- `images/1.png` ... `images/108.png`
- `metadata/1.json` ... `metadata/108.json`

Metadata follows the Stargaze / CosmWasm standard with attributes: `Gender`, `Background`, `Body`, `Mask`.

After uploading images to IPFS, replace `PLACEHOLDER_CID` in metadata files with your real CID:

```bash
sed -i '' 's/PLACEHOLDER_CID/YOUR_CID/g' metadata/*.json
```

## Landing Page

Built with **React + TypeScript + Vite + Tailwind CSS**.

### Tech Stack

| Technology     | Purpose       |
| -------------- | ------------- |
| React 18       | UI framework  |
| TypeScript     | Type safety   |
| Vite 5         | Build tool    |
| Tailwind CSS 3 | Styling       |
| lucide-react   | Icons         |

### Design System

- Dark gradient background (`#050505` → `#2E2E2E`)
- **Liquid Glass** morphism effects (`backdrop-filter: blur`)
- **Anton** font — headings & uppercase labels
- **Condiment** font — cursive accent text
- Gold accent (`#A08965`) · Navy (`#21304E`) · Silver (`#CCCAC6`)
- Grain texture overlay for premium feel
- Animated particle canvas in Hero section

### Sections

1. **Hero** — Particle animation, headline, hero video, CTA buttons
2. **Lore** — Story of DarkStar, NFT showcase with hover effect
3. **Gallery** — Featured entity cards with trait display
4. **Roadmap** — 3-phase timeline: Genesis Mint → The Bridge → zkNFT
5. **Footer** — Social links, copyright

### Run Locally

```bash
cd website

npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

### Build for Production

